# Solve Everything (audiobook files + transcripts)

This repo contains an experimental audiobook rendition of *Solve Everything* by Alexander Wissner-Gross and Peter Diamandis, plus chapter transcripts.

## Credits (prominent)

- Title: *Solve Everything*
